# Password generator app - generates a password with user defined settings with the following characteristics (minimum 8 characteres):
- include upper case letters
- include lower case letters
- include digits
- include symbols
- make the pass somewhat pronounceble by generating sylables is the option is selected
- it informs the user about the strength of the generated pass and give visual indication of the characters that are present in the pass
- logs each generated pass in file
- can be copied to the clipboard
- can be saved to a file
